🌤️ Weather Dashboard App

A simple weather application that allows users to search for real-time weather information for any city using the OpenWeather API. It also stores recent searches for quick access.


🚀 Features
🔍 Search weather by city name
🌡️ View temperature, humidity, wind, and pressure
🌤️ Dynamic weather icons
⚠️ Error handling (invalid cities, API issues)
⏳ Loading indicator
🕒 Recent search history (saved in local storage)
📱 Clean and responsive UI


🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)

🔑 API Setup

This project uses the OpenWeather API.

1. Get your API key
Go to: https://openweathermap.org/
Sign up / Log in
Create an API key
2. Add your API key in app.js
const API_KEY = "YOUR_API_KEY_HERE";


▶️ How to Run the Project
Download or clone the repository
Open index.html in your browser
Enter a city name and search

⚠️ Common Issues
❌ "City not found"
Check spelling
Try a different city
❌ "Failed to fetch"
Check internet connection
Ensure API key is correct and active
